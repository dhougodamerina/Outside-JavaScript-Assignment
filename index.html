About Game: 2048 is a kind of puzzle game which at first contains two tiles that display the number 2 or 4. With the help of the arrow keys one can move the tiles one step at a time. One can generate new tiles by moving or colliding two equal value tiles. When two tiles of the same value collide it generates a value that is twice the two colliding values. The goal of this game is to get 2048 before the whole box fills up with tiles.
